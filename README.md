Generate a `.WAV` audio form using C. Can play "When The Saints Go Marching In" somewhat well!

# Features
- Note-by-note playback
- Polyphony
- Arbitrary note sounds (in Hz)
- Sine generation
    - Global timer based (this may be changed later)
    - Does not support non-sine waveforms
    - Does not support samples

# How To Use
This currently doesn't really support any sort of "user interface" yet, but you could edit the note sequence if you want, recompile, and use the binary to generate a melody.
